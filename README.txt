 Currency Converter GUI Application

This project uses Python and Tkinter to create a currency converter application with a graphical user interface. Users can select two currencies, enter an amount, and convert the value using live exchange rate data from an online API.

Language Used:
Python

Libraries Used:
tkinter
tkinter.ttk
PIL (Pillow)
requests
json

Main Skills Practiced:
- GUI development
- API integration
- JSON data handling
- dropdown menus with Combobox
- buttons and labels
- user input handling
- image handling
- currency conversion logic

Project Features:
- Converts money between multiple currencies
- Uses live exchange rate data from an API
- Interactive GUI with dropdown menus
- Displays converted currency values
- Includes custom colors, styling, and images
- Uses Combobox widgets for currency selection
